# HITO 1 · Setup técnico del proyecto

## Objetivo
Inicializar el proyecto con una base técnica sólida, limpia y escalable para el desarrollo de la landing page.

## Alcance
- Crear proyecto en Next.js (App Router)
- Configurar Tailwind CSS
- Definir estructura base de carpetas
- Configurar estilos globales y tipografías

## Tareas clave
- Inicializar proyecto Next.js
- Instalar y configurar Tailwind
- Crear estructura base (/app, /components, /sections)
- Configurar layout global

## Entregables
- Proyecto funcionando en local
- Tailwind operativo
- Base lista para maquetar

## Criterio de finalización
El proyecto corre sin errores y permite comenzar el desarrollo visual.
